# Monetización

Botanic arranca **100% gratuita**. Los ingresos se activan cuando hay masa crítica.

## Planes

| | Gratis | Pro (Fase 2) | Viveros (Fase 3) |
|---|---|---|---|
| **Precio** | 0 €/mes | ~3,99 €/mes | ~19 €/mes |
| Anuncios | Ilimitados | Ilimitados | Catálogo ilimitado |
| Fotos/anuncio | 5 | 10 | 20 |
| Destacados | — | ✅ | ✅ |
| Aparecer primero | — | ✅ | ✅ |
| Estadísticas | — | ✅ | ✅ |
| Badge verificado | — | ✅ | ✅ |
| Perfil profesional | — | — | ✅ |
| Múltiples usuarios | — | — | ✅ |

## Líneas de ingreso adicionales

1. **Anuncios destacados** (Fase 2): 1 €/día para aparecer arriba
2. **Comisión por transacción** (Fase 3): 3-5% en pagos integrados
3. **Publicidad viveros locales** (Fase 3): aparecer en búsquedas de zona
4. **Suscripción Viveros** (Fase 3): plan mensual profesional

## Prioridad

```
Fase 1 (MVP):      100% gratis — crecer la comunidad
Fase 2 (6-9m):     Destacados + Pro (bajo coste)
Fase 3 (12m+):     Comisiones + Viveros
```
